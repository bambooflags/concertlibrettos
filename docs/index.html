About This Project

Concert Librettos provides free, beautifully formatted program notes for Berliner Philharmonie concerts. Each libretto includes composer biographies, historical context, and insights into the music you're about to experience.

This is a passion project created to enhance the concert-going experience. If you find these librettos helpful, consider buying me a coffee to support the project.

Concert Librettos is an independent project and is not affiliated with the Berliner Philharmonie.
